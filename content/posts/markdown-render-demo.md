## Texts

| Markdown | Rendered |
| - | - |
| `**Bold**` | **Bold** |
| `__Bold__` | __Bold__ |
| `*Italic*` |  *Italic* |
| `_Italic_` |  _Italic_ |
| `~Strikethrough~` |  ~Strikethrough~ |
| `~~Strikethrough~~` |  ~~Strikethrough~~ |

## Blockquotes

Markdown:

```
> Enim anim eiusmod occaecat labore ipsum.
```

Rendered:

> Enim anim eiusmod occaecat labore ipsum.

## Link

Markdown:

```
[https://www.example.com](https://www.example.com)

<address@example.com>
```

Rendered:

[https://www.example.com](https://www.example.com)

<address@example.com>
